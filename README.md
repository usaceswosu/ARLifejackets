# ARLifejackets

ARLifejackets is a Python project for detecting and tracking augmented reality lifejackets using computer vision and machine learning.

## Features

- Real-time detection using YOLO
- Pose/landmark tracking with MediaPipe
- Works with webcam or video input
- TensorFlow Lite CPU support for lightweight inference

## Requirements

This project uses the following Python packages:
pip install -r requirements.txt
- `opencv-python`
- `mediapipe`
- `ultralytics`
- `scipy`
- `pytesseract`
- `pyautogui`
- `numpy`

## Setup!

Step 1: Download the file by clicking the blue code button on the home page of this GitHub and then selecting "Download ZIP".
Step 2: Find the file in your downloads and drag it to your desktop
Step 3: Right-click the File and press Extract All. Once that's done, you can delete the one with a zipper on the file.
Step 4: Open the new file that appeared on your desktop and then open the setup.bat file.
Step 5: Enjoy!
